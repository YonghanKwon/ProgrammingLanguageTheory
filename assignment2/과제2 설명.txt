다음은 C 언어 문법에 따라 작성된 프로그램으로, 인자 전달이 pass-by-value에 따를 경우 1 1을 결과로 출력한다.

 

    int i = 0;

    int a[2] = {1, 1};

    void p(int x, int y) {

        x++; i++; y++;

    }

    main() {

          p(a[i], a[i]);

          printf("%d %d\n", a[0], a[1]);

    }

 

인자 전달 규칙이 다음에 따를 경우를 위한 코드를 작성하고 예상되는 출력을 보여라. (힌트: 모든 규칙의 결과가 서로 다르다.) 

By-reference
By-value-result (호출 시에 반환 주소 확정)
By-value-result (return 시에 반환 주소 확정)
By-name
(주의)

각 코드는 언어의 기본 기능만을 사용하여 인자 전달 규칙을 모방하여야 한다.
C 언어 이외의 다른 언어를 사용할 수도 있으나 모든 코드는 동일한 언어를 사용해야 한다.
1)~4)를 하나의 프로그램 파일로 작성하여 텍스트로 제출한다.
